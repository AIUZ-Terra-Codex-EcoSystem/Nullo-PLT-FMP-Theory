> Theoretical layer for NULLO, PLT, and FMP: conceptual spine, doctrine, and
> the epistemic framework that supports the wider Terra ecosystem.

## Layer role

This repository is a theory layer.

It should hold:

- theoretical foundations of NULLO, PLT, and FMP;
- readable conceptual and doctrinal explanations;
- theory-facing publication and release surfaces;
- a clear human entry into the theory stack.

It should not become:

- a symbolic shell with no reading path;
- a practice repository;
- a donor governance repository;
- a mixed archive of unrelated branch residue.

## Reading path

1. `README.md`
2. `CITATION.cff`
3. theory-facing root files
4. `Publications/`
5. `.github/workflows/`

## Ecosystem position

- `Theory-of-fractal-metascience-paradigm` — related theory paper
- `FMP-CENTRAL-REPO` — central public hub
- `terra-legal` — donor governance and standards
- `Nullo-PLT-UCOMM-FMP-Academic-Research` — academic bridge layer

## Operating rule

This repository must remain readable for humans first.

Its role is to expose theory clearly, not to hide it behind generic protocol
markers and placeholder scaffolding.
